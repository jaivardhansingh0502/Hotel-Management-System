# Hotel Management System

## Overview

Hotel Management System is a console-based application developed in C++ using Object-Oriented Programming principles. It simulates the core functionalities of a hotel, including customer management, room management, room booking, checkout, bill generation, and hotel reporting through a menu-driven interface.

---

## Features

- Add Customer
- Add Room
- Display Customers
- Display Rooms
- Book Room
- Check Out Room
- Generate Bill
- Display Bookings
- Hotel Report

## How to Run

Compile the program:

```bash
g++ HotelManagementgit.cpp -o hotel
```

Run the executable:

```bash
./hotel
```

For Windows PowerShell:

```powershell
.\hotel.exe
```
